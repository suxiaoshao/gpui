# 应用外壳

应用外壳包含主窗口入口、菜单栏、托盘和关于窗口。它负责让用户进入主要功能，而不是处理具体对话内容。

## 子功能

- [菜单栏与托盘](menus-tray.md)：打开主窗口、临时对话、设置、关于和退出。
- [窗口与关于](windows-about.md)：主窗口、设置窗口、临时窗口、消息预览窗口和关于窗口。

## 主要状态

- 主窗口已打开或隐藏。
- 托盘可用时显示常用入口。
- 菜单栏按当前语言显示。
- 关于窗口显示版本和项目入口。

## 边界情况

- macOS 和非 macOS 菜单呈现不同，但功能入口应一致。
- 语言切换后菜单和托盘文案应刷新。
- 关闭主窗口行为应符合平台预期。
