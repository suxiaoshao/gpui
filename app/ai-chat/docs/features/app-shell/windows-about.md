# 应用外壳 - 窗口与关于

## 功能目标

窗口功能负责创建和恢复主窗口、设置窗口、临时对话窗口、消息预览窗口和关于窗口。

## 入口

- 应用启动。
- 菜单栏和托盘。
- 主页、设置、消息操作和临时对话中的窗口入口。

## 主要状态

- 主窗口显示主页。
- 设置窗口显示设置页。
- 临时窗口独立显示临时对话。
- 消息预览窗口显示单条消息。
- 关于窗口显示版本和仓库入口。

## 用户动作

- 打开、关闭、最小化和重新打开窗口。
- 从关于窗口打开项目仓库。

## 边界情况

- 重复打开设置或临时对话应复用合理窗口。
- 关于窗口尺寸应紧凑，不遮挡主要信息。
- 主窗口关闭后是否退出应符合平台行为。
