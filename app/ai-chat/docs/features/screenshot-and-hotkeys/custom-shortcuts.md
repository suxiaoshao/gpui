# 截图与快捷键 - 自定义全局快捷键

## 功能目标

自定义全局快捷键可把模板、模型、对话模式和输入来源绑定到一个热键，用于快速发起请求。

## 入口

- 设置 > 快捷键。

## 主要状态

- 绑定启用。
- 绑定停用。
- 模型不可用。
- 热键冲突。
- 注册失败。
- 运行时等待截图、选中文本或剪贴板。

## 用户动作

- 触发全局快捷键。
- 查看绑定状态。
- 重新注册绑定。
- 编辑或删除绑定。

## 边界情况

- 选中文本、剪贴板和截图输入为空时应有明确处理。
- 模型或模板被删除后，快捷键状态应变为可理解的不可用。
- 运行中再次触发不应产生重复请求污染。
