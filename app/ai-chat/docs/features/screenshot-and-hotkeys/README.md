# 截图与快捷键

截图与快捷键能力用于从系统级快捷键收集上下文，包括打开临时会话、执行自定义快捷键绑定、截图 OCR 或读取选中文本/剪贴板。

## 子功能

- [截图叠层](screenshot-overlay.md)：选择屏幕区域并取消截图。
- [临时会话快捷键](temporary-hotkey.md)：快速打开或切换临时对话。
- [自定义全局快捷键](custom-shortcuts.md)：执行设置中配置的快捷键绑定。

## 主要状态

- 快捷键已注册。
- 快捷键冲突或注册失败。
- 截图叠层打开、拖拽选择、取消。
- 快捷键触发后进入临时对话或发送预设请求。

## 边界情况

- Escape 只取消截图叠层，不应影响其他普通窗口。
- 截图区域不能超出显示器边界。
- 自定义快捷键不能与临时会话快捷键冲突。
