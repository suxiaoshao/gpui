# 主页 - 侧边栏

## 功能目标

侧边栏用于组织会话和文件夹，支持创建、编辑、删除、移动和选择会话。

## 入口

- 打开主窗口后位于左侧。
- 通过侧边栏操作按钮或文件夹/会话上下文菜单操作。

## 主要状态

- 根目录包含顶层文件夹和会话。
- 文件夹可展开或折叠。
- 当前会话有选中态。
- 拖放时显示可放置或不可放置反馈。

## 用户动作

- 新建会话。
- 新建文件夹。
- 编辑会话名称、图标、说明。
- 编辑文件夹名称。
- 将会话或文件夹移动到其他文件夹或根目录。
- 删除会话或文件夹。

## 边界情况

- 删除需要确认。
- 文件夹不能移动到自身或后代中。
- 同一父级内移动到原位置应保持不变。
- 中文标题应可正常显示和搜索。
