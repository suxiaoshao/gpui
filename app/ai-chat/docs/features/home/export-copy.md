# 主页 - 导出与复制会话

## 功能目标

导出用于把会话保存为 JSON、CSV 或 TXT。复制会话用于基于当前会话信息创建一个新会话，但不复制历史消息。

## 入口

- 会话头部导出按钮。
- 会话头部复制按钮。
- 会话上下文菜单中的复制入口。

## 主要状态

- 导出菜单列出可选格式。
- 保存对话框选择路径和文件名。
- 复制会话打开新建会话对话框，并预填当前会话信息。

## 用户动作

- 导出 JSON、CSV、TXT。
- 取消导出。
- 使用相同文件名重复导出。
- 复制会话并提交新建会话表单。

## 边界情况

- 导出不应覆盖已有文件，应生成带后缀的新文件名。
- CSV 需要正确处理逗号、引号和换行。
- 复制会话不应复制消息历史。
