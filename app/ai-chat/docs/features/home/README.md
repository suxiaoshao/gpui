# 主页

主页是 `ai-chat` 的主要工作区，包含会话树、会话搜索、标签页、会话导出/复制和对话面板。

## 子功能

- [侧边栏](sidebar.md)：文件夹和会话树、创建、编辑、移动、删除。
- [会话搜索](search.md)：按名称、说明、文件夹和拼音搜索会话。
- [标签页](tabs.md)：打开多个会话并在会话间切换。
- [导出与复制](export-copy.md)：导出会话或复制为新会话。
- [对话](conversation/README.md)：发送消息、控制生成、消息操作和聊天表单。

## 主要状态

- 空状态：没有会话或没有打开标签页。
- 浏览状态：侧边栏选中一个会话，右侧显示对话。
- 搜索状态：弹出搜索框和结果列表。
- 编辑状态：打开会话、文件夹、消息或导出相关对话框。

## 边界情况

- 文件夹嵌套和拖放不能形成循环。
- 搜索没有结果时应有明确空状态。
- 会话没有消息时，对话区应提示下一步操作。
- 窄窗口下侧边栏、标签页和对话区仍应可操作。
