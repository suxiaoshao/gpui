# 主页 - 会话搜索

## 功能目标

会话搜索用于快速定位会话。它按会话标题、说明、文件夹路径、中文拼音和拼音首字母匹配。

## 入口

- 侧边栏搜索按钮。
- 主窗口聚焦主页时使用搜索快捷键。

## 主要状态

- 初始状态显示搜索输入框。
- 有结果时显示会话和所在路径。
- 无结果时显示空状态。
- 选择结果后打开对应会话标签页。

## 用户动作

- 输入关键词。
- 使用鼠标点击结果。
- 使用键盘上下键和 Enter 选择结果。
- 使用 Escape 关闭搜索。

## 边界情况

- 搜索只在主页上下文触发，不应在设置、关于、临时对话或消息预览中误触发。
- 空格或空查询不应产生误导结果。
