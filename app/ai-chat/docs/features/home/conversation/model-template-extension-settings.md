# 主页 - 对话 - 模型、模板与扩展设置

## 功能目标

模型、模板与扩展设置决定本次请求使用哪个提供方模型、套用哪个提示词模板，以及使用哪些提供方扩展参数。

## 入口

- 聊天表单中的模型选择控件。
- 聊天表单中的模板选择控件或 `/` 触发。
- 模型支持扩展设置时显示的扩展控件。

## 主要状态

- 模型加载中。
- 模型加载失败。
- 模型可选并按提供方分组。
- 模板可搜索和选择。
- 扩展设置根据模型能力显示。

## 用户动作

- 刷新模型列表。
- 选择模型。
- 选择或清除模板。
- 修改扩展设置，例如 reasoning、thinking 或 web search。

## 边界情况

- 当前选择的模型暂时未加载时，应保留用户选择直到能解析。
- 无效的已保存扩展设置不应阻止请求。
- 提供方部分失败时，可用提供方仍应显示模型。
