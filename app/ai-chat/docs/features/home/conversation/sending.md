# 主页 - 对话 - 消息发送

## 功能目标

消息发送把用户输入、模板、模型、模式和扩展设置组合成一次模型请求，并把响应写入当前会话。

## 入口

- 打开任意会话。
- 在底部聊天表单输入内容并提交。

## 主要状态

- 可发送：输入框有内容且模型可用。
- 发送中：用户消息和助手占位消息出现，状态为加载或思考。
- 完成：助手消息变为正常状态。
- 失败：助手消息显示错误。

## 用户动作

- 输入用户消息。
- 选择模式后发送。
- 使用模板生成预填提示。
- 使用提供方扩展设置后发送。

## 边界情况

- 空白输入不能发送。
- 模型不可用时应提示配置或选择其他模型。
- 网络或提供方错误应显示在对应消息上。
