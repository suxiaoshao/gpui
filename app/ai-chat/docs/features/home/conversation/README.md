# 主页 - 对话

对话是普通会话的消息工作区，负责消息发送、模型响应、消息操作和聊天表单。

## 子功能

- [消息发送](sending.md)：按模式构造上下文并发送用户消息。
- [生成控制](generation-controls.md)：暂停、重发、清空会话。
- [消息操作](message-actions.md)：复制、预览、编辑、删除、重新生成。
- [聊天表单](chat-form.md)：输入内容、选择模板、提交或暂停。
- [模型/模板/扩展设置](model-template-extension-settings.md)：模型选择、模板选择和提供方扩展参数。

## 主要状态

- 空会话：等待用户输入第一条消息。
- 正常消息：显示角色、内容和操作。
- 加载/思考中：显示生成状态和暂停入口。
- 错误消息：显示可见错误和重试入口。

## 边界情况

- 提供方未配置时不能静默失败。
- 单轮、上下文、仅助手三种模式应构造不同历史。
- 暂停和重发不能破坏已有消息顺序。
