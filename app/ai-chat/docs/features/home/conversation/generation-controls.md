# 主页 - 对话 - 生成控制

## 功能目标

生成控制用于在模型响应过程中暂停生成，或对已有消息执行重发和清空。

## 入口

- 生成中消息上的暂停按钮。
- 消息操作菜单中的重新发送。
- 会话头部的清空按钮。

## 主要状态

- 生成中：暂停按钮可用。
- 已暂停：消息状态显示为暂停。
- 可重发：用户消息或可重发的助手消息显示重发入口。
- 清空确认：清空会话前显示破坏性确认。

## 用户动作

- 暂停正在生成的消息。
- 重发指定消息。
- 清空当前会话全部消息。

## 边界情况

- 暂停非运行消息不应改变内容。
- 重发应基于正确消息历史，不重复追加旧内容。
- 清空必须确认，取消后消息保留。
