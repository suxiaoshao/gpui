# ai-chat 功能文档

本文档从产品能力出发描述 `ai-chat`，不按代码模块组织。每个功能文档都在 `docs/tests/` 下有对应测试步骤，用于指导人工验证和 agent 使用 `computer-use` 等工具执行 UI 验证。

## 产品结构

- [应用外壳](app-shell/README.md)：主窗口、菜单栏、托盘、关于窗口。
- [主页](home/README.md)：侧边栏、会话搜索、标签页、会话导出/复制、对话工作区。
- [设置](settings/README.md)：通用、外观、提供方、模板、快捷键。
- [消息预览](message-preview/README.md)：查看、编辑、删除、重新生成消息。
- [临时对话](temporary-chat/README.md)：独立临时对话窗口、保存为普通会话、失焦隐藏。
- [截图与快捷键](screenshot-and-hotkeys/README.md)：截图叠层、临时会话快捷键、自定义全局快捷键。

## 核心工作流

1. 用户在主页组织文件夹和会话。
2. 用户选择模型、模板和扩展设置后发送消息。
3. 用户在会话中暂停、重发、预览、编辑或删除消息。
4. 用户通过设置维护语言、主题、提供方、模板和快捷键。
5. 用户可用临时对话或全局快捷键快速发起一次上下文外请求。

## 文档约定

- “入口”描述用户如何到达该功能。
- “主要状态”描述用户可观察到的 UI 状态。
- “用户动作”描述可执行操作，不描述内部实现。
- “边界情况”描述空状态、错误状态、权限/配置缺失、窄窗口和本地化等风险。
