# 设置 - 通用

## 功能目标

通用设置用于管理语言、配置文件入口、HTTP 代理和临时会话快捷键。

## 入口

- 打开设置窗口后选择“通用”。
- 从设置搜索中搜索“语言”“代理”“配置文件”“临时会话快捷键”。

## 主要状态

- 语言可选：跟随系统、English、简体中文。
- 配置文件提供打开入口。
- HTTP 代理为可编辑文本。
- 临时会话快捷键使用热键输入控件。

## 用户动作

- 切换语言并观察应用文案刷新。
- 编辑 HTTP 代理。
- 打开配置文件。
- 设置、清空或修改临时会话快捷键。

## 边界情况

- 无效快捷键不应保存为可用快捷键。
- 切换语言后，已保存配置不应丢失。
- 打开配置文件失败时应显示可理解的错误反馈。
