# 设置

设置用于维护全局偏好、主题、模型提供方、提示词模板和全局快捷键。入口为应用菜单中的“设置”、侧边栏设置按钮，或其他功能跳转到指定设置页。

## 子功能

- [通用](general.md)：语言、配置文件、HTTP 代理、临时会话快捷键。
- [外观](appearance.md)：外观模式、主题选择、自定义 Material You 主题。
- [提供方](provider.md)：OpenAI、Ollama 等模型提供方的连接配置。
- [模板](templates.md)：可复用提示词模板的列表、搜索、查看、创建、编辑、删除。
- [快捷键](shortcuts.md)：绑定模板、模型、模式和输入来源的全局快捷键。

## 主要状态

- 默认状态：左侧显示设置导航和搜索，右侧显示当前页。
- 搜索状态：只展示匹配的设置页或设置项。
- 空/错误状态：模板、快捷键或模型加载失败时显示可恢复的提示和操作。

## 边界情况

- 切换语言后，当前设置窗口、菜单和相关页面应刷新可见文本。
- 窄窗口下，输入框、表格和按钮不应被遮挡。
- API Key 等敏感配置默认不应明文显示。
