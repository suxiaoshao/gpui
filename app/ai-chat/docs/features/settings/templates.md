# 设置 - 模板

## 功能目标

模板设置用于管理可复用提示词。模板可在普通会话、临时会话和快捷键绑定中复用。

## 入口

- 打开设置窗口后选择“模板”。
- 从设置搜索中搜索“模板”“提示词”“添加”“编辑”“删除”。

## 主要状态

- 列表展示模板图标、名称、描述和 prompt 数量。
- 搜索可按名称、描述、中文拼音和首字母匹配。
- 查看对话框展示模板元数据和 prompt 列表。
- 添加/编辑使用同一类表单。

## 用户动作

- 搜索模板。
- 查看模板详情。
- 添加模板和多个 prompt。
- 编辑模板名称、图标、描述和 prompt。
- 删除模板。

## 边界情况

- 名称或图标为空时不能保存。
- prompt 缺少角色或内容时不能保存。
- 删除模板需要破坏性确认。
- 模板被快捷键引用时，列表和快捷键状态应能表达可用性。
