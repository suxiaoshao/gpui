# 设置 - 快捷键

## 功能目标

快捷键设置用于创建全局快捷键绑定。绑定可指定模板、模型、对话模式、发送内容和输入来源。

## 入口

- 打开设置窗口后选择“快捷键”。
- 从设置搜索中搜索“快捷键”“Hotkey”“模板”“模型”“截图”“OCR”。

## 主要状态

- 列表展示模板、热键、输入来源、模型、模式、状态、启用状态和操作。
- 空状态提供添加入口。
- 加载失败状态提供重新加载入口。
- 状态详情说明模型不可用、热键冲突、注册失败等问题。

## 用户动作

- 添加快捷键绑定。
- 编辑模板、模型、模式、发送内容、热键和启用状态。
- 查看状态详情。
- 重新注册快捷键。
- 删除快捷键。

## 边界情况

- 临时会话快捷键和自定义绑定不能冲突。
- 当前绑定编辑时不应误报自身冲突。
- 模型列表未加载或模型被删除时，应显示可恢复状态。
