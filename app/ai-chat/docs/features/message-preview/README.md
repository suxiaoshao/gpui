# 消息预览

消息预览用于查看或编辑单条普通会话消息和临时会话消息。它把消息元数据、正文、请求内容和操作集中在独立窗口中。

## 子功能

- [预览](preview.md)：查看消息内容、角色、状态、时间和请求信息。
- [编辑](edit.md)：修改可编辑消息正文。
- [删除与重新生成](destructive-actions.md)：删除消息或重新生成消息。
- [元数据和内容展示](metadata-content.md)：展示消息 ID、角色、状态、引用、请求体等信息。

## 主要状态

- 预览模式：内容只读。
- 编辑模式：可编辑字段清晰可见。
- 保存中/错误状态：保存失败时显示反馈。
- 窄窗口状态：内容可滚动，主要操作可达。

## 边界情况

- 空引用或空请求内容不应占据过多空间。
- 加载、暂停、错误状态应显示可理解状态。
- 删除和重新生成需要确认。
