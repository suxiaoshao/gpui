# 临时对话

临时对话用于快速发起不一定进入普通会话树的请求。用户可以在需要时把临时对话保存为普通会话。

## 子功能

- [入口](entry.md)：从菜单、托盘或全局快捷键打开。
- [消息交互](messaging.md)：选择模板、发送、暂停、预览和重发。
- [保存、清空和失焦隐藏](save-clear-hide.md)：保存为普通会话、清空、窗口失焦行为。

## 主要状态

- 空状态：说明可直接提问或选择模板。
- 对话状态：显示临时消息列表和聊天表单。
- 生成中状态：可暂停。
- 保存状态：打开新建会话对话框并预填消息。

## 边界情况

- 重复触发临时会话快捷键应复用窗口。
- 空消息时清空和保存不应误导用户。
- 失焦隐藏不应丢失正在生成的状态。
