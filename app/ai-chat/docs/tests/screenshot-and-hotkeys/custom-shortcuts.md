# 截图与快捷键 - 自定义全局快捷键测试

## 用例：新增自定义快捷键

### 测试目标

验证用户可在测试配置中新增一个自定义全局快捷键。

### 数据隔离

只写入测试配置目录。不要注册会影响用户真实工作流的快捷键。

### 测试前提

- 应用已使用测试配置启动。
- 设置窗口已打开到 `快捷键` 页面。

### 测试数据

- 动作名称：`打开主窗口`。
- 快捷键：`Cmd+Shift+7`。

### 测试步骤

1. 点击 `新增快捷键` 按钮。
2. 在动作选择控件中选择 `打开主窗口`。
3. 在快捷键输入框按下 `Cmd+Shift+7`。
4. 点击保存按钮。
5. 关闭设置窗口。
6. 按下 `Cmd+Shift+7`。

### 预期结果

- 快捷键列表新增 `打开主窗口` - `Cmd+Shift+7`。
- 第 6 步主窗口打开或聚焦。
- 不修改用户真实快捷键配置。

### 边缘情况

- 动作未选择时保存按钮应禁用，或点击后显示动作必填错误。

### 清理

删除 `Cmd+Shift+7` 测试快捷键绑定，关闭应用并删除测试配置目录。

## 用例：编辑自定义快捷键

### 测试目标

验证已有测试快捷键可被修改。

### 数据隔离

只编辑测试快捷键，不读取用户真实快捷键。

### 测试前提

- 测试配置中已有 `打开主窗口` - `Cmd+Shift+7`。
- 设置窗口已打开到 `快捷键` 页面。

### 测试数据

- 原快捷键：`Cmd+Shift+7`。
- 新快捷键：`Cmd+Shift+6`。

### 测试步骤

1. 在快捷键列表中找到 `打开主窗口` - `Cmd+Shift+7`。
2. 点击该行编辑按钮。
3. 在快捷键输入框中按下 `Cmd+Shift+6`。
4. 点击保存按钮。
5. 观察快捷键列表。

### 预期结果

- 列表中不再显示 `Cmd+Shift+7`。
- 列表中显示 `打开主窗口` - `Cmd+Shift+6`。
- 按下 `Cmd+Shift+6` 可打开或聚焦主窗口。

### 边缘情况

- 编辑后按下旧快捷键 `Cmd+Shift+7` 不应再触发打开主窗口。

### 清理

删除 `Cmd+Shift+6` 测试快捷键绑定，删除测试配置目录。

## 用例：删除自定义快捷键时取消和确认

### 测试目标

验证删除快捷键的取消与确认路径。

### 数据隔离

只删除测试快捷键。

### 测试前提

- 测试配置中已有 `打开主窗口` - `Cmd+Shift+6`。
- 设置窗口已打开到 `快捷键` 页面。

### 测试数据

- 删除目标：`打开主窗口` - `Cmd+Shift+6`。

### 测试步骤

1. 在快捷键列表中找到 `打开主窗口` - `Cmd+Shift+6`。
2. 点击该行删除按钮。
3. 在确认对话框点击取消按钮。
4. 确认列表仍显示 `打开主窗口` - `Cmd+Shift+6`。
5. 再次点击该行删除按钮。
6. 在确认对话框点击确认删除按钮。
7. 观察快捷键列表。

### 预期结果

- 第 3 步取消后快捷键仍存在。
- 第 6 步确认后快捷键从列表中移除。
- 按下 `Cmd+Shift+6` 不再触发打开主窗口。

### 边缘情况

- 重复点击确认删除按钮不应删除其他快捷键。

### 清理

关闭应用并删除测试配置目录。
