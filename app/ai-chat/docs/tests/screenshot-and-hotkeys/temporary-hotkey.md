# 截图与快捷键 - 临时会话快捷键测试

## 用例：设置并触发临时会话快捷键

### 测试目标

验证自定义临时会话快捷键可保存并触发临时对话。

### 数据隔离

只在测试配置目录中注册快捷键 `Cmd+Shift+9`，不覆盖用户真实全局快捷键。

### 测试前提

- 应用已使用测试配置启动。
- `Cmd+Shift+9` 未被测试环境中其他应用占用。

### 测试数据

- 快捷键：`Cmd+Shift+9`。

### 测试步骤

1. 打开设置窗口。
2. 点击左侧导航中的 `快捷键`。
3. 找到 `打开临时对话` 快捷键项。
4. 点击该项的编辑按钮。
5. 在快捷键输入框按下 `Cmd+Shift+9`。
6. 点击保存按钮。
7. 关闭设置窗口。
8. 按下 `Cmd+Shift+9`。

### 预期结果

- 快捷键列表显示 `打开临时对话` 的绑定为 `Cmd+Shift+9`。
- 第 8 步打开临时对话窗口。
- 不修改用户真实快捷键配置。

### 边缘情况

- 如果系统拒绝注册该快捷键，应显示明确错误，设置窗口不应把失败快捷键显示为已生效。

### 清理

删除测试快捷键绑定，关闭临时对话窗口，删除测试配置目录。

## 用例：临时会话快捷键冲突

### 测试目标

验证快捷键冲突时阻止保存或显示冲突错误。

### 数据隔离

只在测试配置中制造冲突，不读取用户真实快捷键。

### 测试前提

- 测试配置中已有 `截图` 快捷键绑定为 `Cmd+Shift+8`。
- 设置窗口已打开到 `快捷键` 页面。

### 测试数据

- 冲突快捷键：`Cmd+Shift+8`。

### 测试步骤

1. 找到 `打开临时对话` 快捷键项。
2. 点击编辑按钮。
3. 在快捷键输入框按下 `Cmd+Shift+8`。
4. 观察字段错误和保存按钮。
5. 如果保存按钮仍可点击，点击保存按钮。

### 预期结果

- 页面显示快捷键冲突提示。
- 保存按钮禁用，或点击后显示冲突错误。
- `打开临时对话` 原快捷键不变。
- `截图` 快捷键仍为 `Cmd+Shift+8`。

### 边缘情况

- 大小写或左右修饰键差异不应绕过同一快捷键冲突检查。

### 清理

取消编辑，删除测试快捷键配置。
