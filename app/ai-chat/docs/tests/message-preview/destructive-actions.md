# 预览 Message - 删除与重新生成测试

## 用例：删除消息时点击取消

### 测试目标

验证删除确认对话框取消后不删除消息。

### 数据隔离

只对测试消息执行删除取消，不操作用户真实消息。

### 测试前提

- 会话 `QA 命名助手` 中存在助手消息 `QA 助手响应内容`。
- 已打开该消息的预览窗口。

### 测试数据

- 目标消息：`QA 助手响应内容`。

### 测试步骤

1. 在预览窗口点击删除按钮。
2. 在删除确认对话框中点击取消按钮。
3. 观察预览窗口。
4. 回到主页消息列表。

### 预期结果

- 删除确认对话框关闭。
- 预览窗口仍显示 `QA 助手响应内容`。
- 主页消息列表仍包含该消息。

### 边缘情况

- 按 `Escape` 关闭删除确认对话框时，应等同于取消。

### 清理

关闭预览窗口，关闭应用并删除测试数据库。

## 用例：删除消息时点击确认

### 测试目标

验证确认删除只删除目标测试消息。

### 数据隔离

测试数据库中至少包含一条相邻消息，用于确认不会误删其他消息。

### 测试前提

- 会话 `QA 命名助手` 中存在用户消息 `请用一句话回复 QA 测试消息`。
- 同一会话中存在助手消息 `QA 助手响应内容`。
- 已打开助手消息预览窗口。

### 测试数据

- 删除目标：`QA 助手响应内容`。
- 保留消息：`请用一句话回复 QA 测试消息`。

### 测试步骤

1. 在预览窗口点击删除按钮。
2. 在删除确认对话框中点击确认删除按钮。
3. 回到主页消息列表。
4. 查找 `QA 助手响应内容`。
5. 查找 `请用一句话回复 QA 测试消息`。

### 预期结果

- `QA 助手响应内容` 从消息列表中移除。
- `请用一句话回复 QA 测试消息` 仍保留。
- 删除后预览窗口关闭或显示消息已删除状态。

### 边缘情况

- 重复点击确认删除按钮不应造成重复删除错误弹窗或删除其他消息。

### 清理

关闭应用并删除测试数据库。

## 用例：重新生成消息时取消

### 测试目标

验证重新生成确认取消后不发起生成。

### 数据隔离

使用测试模型或 mock provider，不能调用真实付费模型。

### 测试前提

- 会话 `QA 命名助手` 中存在可重新生成的助手消息。
- 测试提供方配置为不可真实联网或 mock 响应。

### 测试数据

- 目标消息：`QA 助手响应内容`。

### 测试步骤

1. 打开 `QA 助手响应内容` 的预览窗口。
2. 点击重新生成按钮。
3. 在确认对话框中点击取消按钮。
4. 观察主页消息列表。

### 预期结果

- 不出现新的生成中状态。
- 消息列表仍只有原助手消息。
- 不触发真实网络请求。

### 边缘情况

- 按 `Escape` 关闭确认对话框时，应等同于取消。

### 清理

关闭预览窗口，关闭应用并删除测试数据库。
