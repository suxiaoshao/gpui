# 设置 - 快捷键测试

## 添加快捷键绑定

### 测试目标

验证用户可以新增启用的全局快捷键绑定。

### 数据隔离

用测试数据库预置模板和模型配置。不要注册会影响用户日常使用的快捷键；若冲突，换用测试环境可用组合。

### 测试前提

- 设置窗口已打开并进入“快捷键”页。
- 测试数据库中存在模板 `QA 单提示词模板`。
- 模型列表中至少有一个可选模型。

### 测试数据

- 模板：`QA 单提示词模板`
- 热键：`Cmd+Shift+9`
- 模式：`上下文模式`
- 输入来源：`剪贴板` 或默认可选项

### 测试步骤

1. 点击“添加快捷键”按钮。
2. 在“模板”下拉框选择 `QA 单提示词模板`。
3. 在“模型”下拉框选择第一个可用模型。
4. 在“模式”下拉框选择 `上下文模式`。
5. 在“输入来源”下拉框选择 `剪贴板`；如果没有该选项，选择默认可用输入来源并记录。
6. 聚焦“快捷键”输入框。
7. 按下 `Cmd+Shift+9`。
8. 确认“启用”开关为开启。
9. 点击“保存快捷键”按钮。

### 预期结果

- 对话框关闭。
- 快捷键列表出现 `QA 单提示词模板`。
- 热键列显示 `Cmd+Shift+9` 或等价本地化格式。
- 状态列显示启用、已注册，或可理解的待处理状态。

### 边缘情况

- 如果 `Cmd+Shift+9` 被系统占用，应显示注册失败或冲突，不应静默成功。

### 清理

删除刚创建的快捷键绑定。

## 漏填快捷键必填项

### 测试目标

验证快捷键表单缺少模板、模型或热键时不能保存。

### 数据隔离

只使用测试数据库，不保存无效绑定。

### 测试前提

- 设置窗口已打开并进入“快捷键”页。

### 测试数据

- 热键：`Cmd+Shift+9`

### 测试步骤

1. 点击“添加快捷键”按钮。
2. 保持“模板”为空。
3. 选择第一个可用模型。
4. 聚焦“快捷键”输入框并按下 `Cmd+Shift+9`。
5. 点击“保存快捷键”按钮。
6. 选择模板 `QA 单提示词模板`。
7. 清空或取消“模型”选择。
8. 点击“保存快捷键”按钮。
9. 选择第一个可用模型。
10. 清空“快捷键”输入框。
11. 点击“保存快捷键”按钮。

### 预期结果

- 第 5、8、11 步均不能保存有效绑定。
- 对话框保持打开，显示必填提示或保存按钮不可用。
- 快捷键列表中不出现不完整绑定。

### 边缘情况

- 已填写的其他字段不应因校验失败丢失。

### 清理

确认未创建无效绑定；若误创建，删除。

## 快捷键冲突

### 测试目标

验证两个绑定不能使用同一个全局热键。

### 数据隔离

仅在测试数据库中创建两个 QA 绑定。

### 测试前提

- 已存在一个使用 `Cmd+Shift+8` 的测试绑定。
- 设置窗口已打开并进入“快捷键”页。

### 测试数据

- 冲突热键：`Cmd+Shift+8`

### 测试步骤

1. 点击“添加快捷键”按钮。
2. 选择模板 `QA 单提示词模板`。
3. 选择第一个可用模型。
4. 聚焦“快捷键”输入框。
5. 按下 `Cmd+Shift+8`。
6. 点击“保存快捷键”按钮。

### 预期结果

- 新绑定不能保存为启用状态。
- UI 显示热键冲突、注册失败或明确错误。
- 已存在的原绑定不被覆盖。

### 边缘情况

- 编辑原绑定时不应把自身识别为冲突。

### 清理

删除所有 `Cmd+Shift+8` 和 `Cmd+Shift+9` 测试绑定。

## 编辑和删除快捷键绑定

### 测试目标

验证快捷键绑定可以编辑，删除支持取消和确认。

### 数据隔离

用命令行预置一个 QA 快捷键绑定。

### 测试前提

- 列表中存在使用 `Cmd+Shift+9` 的 `QA 单提示词模板` 绑定。

### 测试数据

- 新热键：`Cmd+Shift+8`

### 测试步骤

1. 在快捷键列表找到 `QA 单提示词模板`。
2. 点击该行“编辑”按钮。
3. 聚焦“快捷键”输入框。
4. 按下 `Cmd+Shift+8`。
5. 点击“保存快捷键”按钮。
6. 在列表找到更新后的绑定。
7. 点击该行“删除”按钮。
8. 在确认对话框点击“取消”。
9. 确认绑定仍存在。
10. 再次点击“删除”按钮。
11. 在确认对话框点击“删除”。

### 预期结果

- 保存后热键显示为 `Cmd+Shift+8` 或等价格式。
- 取消删除后绑定仍存在。
- 确认删除后绑定从列表消失。

### 边缘情况

- 删除后触发旧热键不应再执行该绑定。

### 清理

确认测试绑定已删除。
