# 应用外壳 - 标题栏与页面切换

## 功能目标

用户可以在主窗口标题栏中识别当前应用和当前页面，并在“高级检索”和“数据抓取”之间切换。

## 入口

- 主窗口标题栏。
- 路由标签：`高级检索`、`数据抓取`。
- 页面级按钮：查询页中的 `抓取`，抓取页中的 `去查询`。

## 主要状态

- 当前页面为 `高级检索`：标题栏显示查询相关摘要，例如条件数、排序数、结果数或搜索状态。
- 当前页面为 `数据抓取`：标题栏显示抓取相关摘要，例如未开始、正在抓取、已中断、抓取失败或抓取成功。
- 查询运行中时，查询按钮显示加载状态并禁用重复查询。
- 抓取运行中时，抓取按钮禁用，抓取页配置输入不可编辑。

## 用户可执行动作

- 点击 `高级检索` 标签进入查询页。
- 点击 `数据抓取` 标签进入抓取页。
- 在查询页点击 `抓取` 进入抓取页。
- 在抓取页点击 `去查询` 进入查询页。

## 重要边界情况

- 页面切换不能丢失当前抓取任务状态。
- 页面切换不能清空未保存的查询条件，除非用户点击 `重置`。
- 抓取运行中切到查询页时，抓取任务应继续运行，标题栏摘要应反映运行状态。
