# 高级检索 - 从抓取页进入查询

## 功能目标

用户抓取完成或需要查看已有数据时，可以从数据抓取页直接进入高级检索。

## 入口

- `数据抓取` 页面中的 `去查询` 按钮。
- 标题栏中的 `高级检索` 标签。

## 主要状态

- 抓取未开始：仍可进入查询页查看已有测试或历史数据。
- 抓取运行中：进入查询页后抓取继续运行。
- 抓取成功：进入查询页后可立即搜索新入库数据。
- 抓取失败：进入查询页不会清空失败状态，用户回到抓取页仍能看到失败页和重试入口。

## 用户可执行动作

- 在抓取页点击 `去查询`。
- 在查询页点击 `抓取` 返回抓取页。
- 在抓取完成后点击 `搜索` 查看入库结果。

## 重要边界情况

- 页面切换不能误触发新的抓取任务。
- 页面切换不能自动清空高级检索条件。
- 页面切换不能把测试数据库切换成用户真实数据库。
