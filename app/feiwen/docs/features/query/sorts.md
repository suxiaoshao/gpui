# 高级检索 - 排序规则

## 功能目标

用户可以按一个或多个具体字段设置排序优先级。

## 入口

- `高级检索` 页面右侧排序区域。

## 主要状态

- 无排序：按默认数据库顺序或默认查询顺序展示。
- 单排序：列表中一条排序规则。
- 多排序：排序列表顺序代表优先级，第一条优先级最高。
- 排序字段为空：显示 `请选择排序字段`。

## 用户可执行动作

- 点击添加排序按钮。
- 在排序字段中选择 `标题`、`作者名称`、`作品 ID`、`最新章节 ID`、`最新章节标题`、`字数`、`阅读数`、`回复数`、`作者 ID`、`是否受限`。
- 在方向中选择 `升序` 或 `降序`。
- 拖拽排序项调整优先级。
- 删除排序项。

## 重要边界情况

- 排序字段为空时不能执行有效查询。
- 拖拽到自身时顺序不变。
- 删除排序项只影响该项，不清空筛选条件。
- 排序规则不能暴露表达式、常量或数学运算。
