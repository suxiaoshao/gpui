# 高级检索 - 结果表格

## 功能目标

用户点击搜索后，在结果表格中查看符合条件的小说列表。

## 入口

- `高级检索` 页面右侧结果表格。
- `搜索` 按钮。

## 主要状态

- 未查询或无结果：显示 `暂无查询结果`。
- 查询中：标题栏或搜索按钮显示正在搜索。
- 有结果：表格展示结果行。
- 查询失败：显示查询失败或错误详情。

## 用户可执行动作

- 点击 `搜索` 执行查询。
- 点击 `重置` 清空条件和排序。
- 查看列：`标题`、`作者`、`字数`、`阅读`、`回复`、`受限`、`最新章节`、`标签`。
- 使用表格滚动查看长结果。

## 重要边界情况

- 空数据库查询应显示无结果，不显示错误。
- 结果较多时表格滚动不应遮挡固定列。
- 标签过多时应在单元格内截断或换行，不应撑破表格。
- 查询失败后用户应能调整条件并重新搜索。
