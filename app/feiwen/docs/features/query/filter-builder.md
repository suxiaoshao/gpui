# 高级检索 - 条件构建器

## 功能目标

用户可以用字段、条件、值和排除开关构造查询条件，表达标题、作者、标签、数字范围和布尔状态等筛选意图。

## 入口

- `高级检索` 页面左侧查询构建器。
- `添加条件`、`添加子组` 按钮。

## 主要状态

- 空组：显示 `添加条件或子组开始构建高级检索。`
- 字段为空：字段控件显示 `请选择字段`，条件和值区域不可用。
- 条件为空：条件控件显示 `请选择条件`，值区域不可用。
- 文本值：显示 `输入文本` 输入框。
- 数字值：显示 `输入数字` 输入框。
- 作者值：显示 `选择作者` 选择器。
- 标签值：显示 `选择标签` 多选器。
- 布尔值：显示是否受限的选项。

## 用户可执行动作

- 点击 `添加条件` 新增条件行。
- 在字段控件选择 `标题`、`简介`、`最新章节标题`、`作者`、`标签`、`字数`、`阅读数`、`回复数`、`是否受限`。
- 在条件控件选择对应字段允许的条件。
- 在值输入器填写文本、数字、范围，或选择作者/标签。
- 开启或关闭条件行 `排除`。
- 点击 `添加子组` 创建嵌套条件组。
- 切换条件组关系 `全部满足` / `任一满足`。
- 开启或关闭 `排除本组`。
- 删除条件行或子组。

## 重要边界情况

- 字段变化必须清空原条件和值，避免旧值被错误复用。
- 条件变化必须清空原值，并切换到正确值输入器。
- 文本值为空时显示 `请输入文本`。
- 数字为空时显示 `请输入数字`。
- 标签或作者集合为空时显示 `请选择至少一项`。
- 无效作者应显示 `请选择有效作者`。
- 空子组需要按当前查询语义处理，不能导致应用崩溃。
