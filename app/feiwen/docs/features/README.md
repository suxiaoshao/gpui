# feiwen 功能文档

feiwen 是面向小说数据的本地管理工具。用户先在“数据抓取”中从站点分页抓取小说、作者、标签和章节摘要，再在“高级检索”中通过结构化条件、排序和结果表格查找目标作品。

本文档按产品能力组织，不按 Rust 模块组织。每个功能文档都在 `../tests/` 下有对应测试步骤文档。

## 产品结构

- [应用外壳](app-shell/README.md)：主窗口标题栏、页面切换、全局操作入口。
- [数据抓取](fetch/README.md)：抓取配置、任务状态、恢复操作、页面日志。
- [高级检索](query/README.md)：条件构建、排序规则、结果表格，以及从抓取页进入查询页。

## 核心工作流

1. 用户打开 feiwen，默认看到“数据抓取”或“高级检索”页面。
2. 用户在“数据抓取”填写入口链接、页码范围和 Cookie，启动分页抓取。
3. 抓取完成后，小说数据进入本地 SQLite 数据库。
4. 用户切换到“高级检索”，添加条件和排序，点击“搜索”。
5. 用户在结果表格中查看标题、作者、字数、阅读、回复、受限状态、最新章节和标签。

## 数据安全原则

feiwen 的抓取和查询都操作本地数据库。产品文档默认描述真实用户能力；测试文档必须使用独立测试数据库，不能抓取到用户真实库，也不能记录真实 Cookie。
