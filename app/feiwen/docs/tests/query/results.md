# 高级检索 - 结果表格测试

## 用例：空数据库搜索显示无结果

### 测试目标

验证测试数据库为空时，搜索显示无结果而不是错误。

### 数据隔离

使用全新的空测试数据库，不连接用户真实数据库。

### 测试前提

- app 尚未启动。
- 测试数据库已经迁移完成但没有小说记录。

### 测试数据

- 按钮：`搜索`
- 空状态文案：`暂无查询结果`

### 测试步骤

1. 使用空测试数据库启动 app。
2. 打开 `高级检索` 页面。
3. 不添加任何条件和排序。
4. 点击 `搜索` 按钮。
5. 观察结果表格区域。

### 预期结果

- 结果表格显示 `暂无查询结果`。
- 标题栏显示 `暂无结果` 或 `0 条结果`。
- 不显示数据库错误。

### 边缘情况

- 点击 `重置` 后空状态保持可理解，不应出现旧结果。

### 清理

关闭 app，删除空测试数据库。

## 用例：结果表格展示所有关键列

### 测试目标

验证查询结果表格展示小说关键字段。

### 数据隔离

使用测试数据库预置 `QA 星河测试小说`，不读取用户真实小说。

### 测试前提

- 测试数据库已预置 `QA 星河测试小说`。
- 该小说有关联作者、最新章节和两个标签。

### 测试数据

- 小说标题：`QA 星河测试小说`
- 作者：`QA 作者甲`
- 最新章节：`QA 最新章节标题`
- 标签：`QA 标签玄幻`、`QA 标签完结`

### 测试步骤

1. 使用测试数据目录启动 app。
2. 打开 `高级检索` 页面。
3. 点击 `搜索` 按钮。
4. 在结果表格中找到 `QA 星河测试小说` 行。
5. 横向查看该行的作者、字数、阅读、回复、受限、最新章节和标签列。

### 预期结果

- 标题列显示 `QA 星河测试小说`。
- 作者列显示 `QA 作者甲`。
- 字数列显示 `120000` 或等价格式化值。
- 阅读列显示 `3456` 或等价格式化值。
- 回复列显示 `78` 或等价格式化值。
- 受限列显示否定状态。
- 最新章节列显示 `QA 最新章节标题`。
- 标签列显示 `QA 标签玄幻` 和 `QA 标签完结`。

### 边缘情况

- 标签列宽不足时，标签应截断、换行或通过滚动查看，不能遮挡后续列。

### 清理

关闭 app，删除测试数据库。

## 用例：查询失败后可重新搜索

### 测试目标

验证一次无效查询失败后，用户修正条件可以重新搜索。

### 数据隔离

使用测试数据库，不修改用户真实数据。

### 测试前提

- 测试数据库已预置 `QA 星河测试小说`。
- app 已打开 `高级检索` 页面。

### 测试数据

- 无效数字：`abc`
- 有效数字：`100000`

### 测试步骤

1. 点击 `添加条件` 按钮。
2. 字段选择 `字数`。
3. 条件选择 `大于`。
4. 在值输入框填写 `abc`。
5. 点击 `搜索` 按钮。
6. 观察条件行错误。
7. 全选值输入框内容并填写 `100000`。
8. 点击 `搜索` 按钮。
9. 观察结果表格。

### 预期结果

- 第 5 步后显示 `请输入有效数字` 或等价数字错误。
- 错误发生后页面保持可编辑。
- 第 8 步后查询成功。
- 结果表格显示 `QA 星河测试小说`。

### 边缘情况

- 修正输入后旧错误提示应消失，不能继续阻止有效搜索。

### 清理

点击 `重置`，关闭 app，删除测试数据库。
